# Automatic Tab Sorting

Moves the active tab to the first unpinned position.
Most recently used tabs always stay on the left.

[![Chrome Web Store](./images/chrome-web-store-install.svg)](https://chromewebstore.google.com/detail/eedheclloenjmpoaiecnfclmbpahbgdg)
[![License: MIT](./images/License-MIT-yellow.svg)](LICENSE)

<img src="./store-images/screenshot-1280x800.png" alt="Screenshot" width="600">

</div>

---

**Features** — zero config, lightweight service worker, pinned tabs unaffected, drag operations preserved, no data collection.

## Build

Requires `librsvg`, `optipng`, `imagemagick`.

```
make store
```
